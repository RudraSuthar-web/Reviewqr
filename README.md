# ReviewQR - AI Powered Review Assistant

A responsive React platform that allows customers to scan a QR code, pick an AI-generated review, and post it directly to Google Maps.

## Tech Stack
- **Framework:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React

## Setup
1. Clone the repo
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`

## Dynamic Usage
To view different business profiles, use the `id` query parameter:
- `localhost:5173/?id=cafe-1`
